Features

User and Role Management
1. Registration via MS, Google and Local Account
2. Email notification on registration, change password, etc. via SMTP (Microsoft Account)
3. Activate User
4. Viewing of Users
5. Updating user details (Roles)

Main Module
1. Add Record
2. Delete Record
3. Update Record
4. View Record

Other Features
1. Pop-up modals are draggable
2. Listings have sorting, paging, search
3. CRUD Functionalities are via ajax (asynchronous)
